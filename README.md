# feathers-index-api

This API is a complement to the Feathers mobile app. It is an index of different bird species and contains information on how to conserve bird populations. 

## Models

**Bird** (/bird/:name)

- [x] speciesName
- [x] name
- [x] sageid
- [x] habitat
- [x] plumage
- [x] nest
- [x] diet
- [x] lifespan
- [x] conservationStatus
- [x] funFacts

**Resource** (/resource)

- [x] name
- [x] url

**Threat** (/threat)

- [x] name
- [x] description

**Tip** (/tip)

- [x] name
- [x] description
